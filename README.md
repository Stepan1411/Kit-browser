# Kits Browser

Web interface for browsing, liking, downloading, and managing PvP Bot kits.

## Features

- Web-based kit browser with inventory preview
- Like and download tracking per user
- Sorting by likes, downloads, or recent
- In-game login/logout via Minecraft commands
- Register/login on the web

## Commands

| Command | Description |
|---------|-------------|
| `/pvpbot kit login` | Generate a login link for the web interface |
| `/pvpbot kit log-out` | Log out of the web interface |
| `/pvpbot kit download-kit <ID>` | Download a kit by ID |
| `/pvpbot kit stats` | Show account info |
| `/pvpbot kit publish-kit <name>` | Publish a kit to the web |
